# 📱 Usability Testing & Analysis

## 📌 Overview

This project demonstrates the usability testing process for **TaskFlow**, a mobile task management application prototype. The objective is to evaluate how users interact with the application, identify usability issues, collect feedback, and suggest design improvements.

---

## 🎯 Objective

- Prepare a usability test plan
- Create test scenarios
- Conduct usability testing
- Record observations and user feedback
- Analyze findings
- Recommend design improvements

---

## 📱 Prototype Screens

The usability testing was conducted using the following prototype screens:

- Splash Screen
- Login Screen
- Home Dashboard
- Add New Task
- Completed Tasks

## 🧪 Usability Testing Summary

The usability testing focused on common user tasks including:

- Launching the application
- Logging into the application
- Viewing daily tasks
- Creating a new task
- Saving a task
- Viewing completed tasks
- Returning to the Home screen

Users completed the assigned tasks successfully, and feedback was collected to identify opportunities for improving the overall user experience.

---

## 💡 Design Improvements Suggested

- Add confirmation messages after saving tasks
- Improve reminder visibility
- Add task search functionality
- Introduce task filtering options
- Enhance profile and settings features
